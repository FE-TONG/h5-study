CSS3~~手风琴


手风琴，不管布局怎么变，主要思想还是首先让整组元素的宽度(高度)都变窄，然后同时被选中的那一个元素的宽度(高度)变宽。

1、竖直ֱ

图为盒子box包裹的一组ul li元素，垂直排列；(实现步骤 如下：)

(1)当鼠标移到这组元素上时，每个li元素 高度统一变窄为30px，
即 ul:hover li{ height:30px; }
(2)同时鼠标移到某一个元素上时，当前这个li元素 高度独自变宽为180px，即 ul li:hover{ height:180px; }

其他布局：每个li之间的间隔为头部内置一个div(颜色不同)，
其box-shadow: inset 0 0 0 5px #112233;


过渡：ul li{ transition: height 0.5s ease-in-out;}

2、水平

图为盒子wrap包裹的一组ul li元素，水平左浮动；(实现步骤 如下：)

(1)当鼠标移到这组元素上时，每个li元素 宽度统一变窄为40px，
即 ul:hover li{ width:40px; }
(2)同时鼠标移到某一个元素上时，当前这个li元素宽度独自变宽为480px，即 ul li:hover{ width:480px; }


过渡：ul li{ transition: width 0.5s;}
