CSS3~~翻转卡片


Y轴旋转：rotateY(< angle >) ;

3D呈现：transform-style: flat | preserve-3d ;

隐藏背面：backface-visibility: visible | hidden ;

过渡：transition: all 时间 ;

透视：perspective: < length > ; / transform: perspective( length ) ;


1.翻转卡片

分析：

1、

将正反两面集中于盒子card中，
重叠在一块

2、

正面翻转0deg，反面翻转180deg。

正反两面都隐藏背面
backface-visibility: hidden;

3、

元素3D空间呈现，即：
transform-style: preserve-3d;

4、

翻转整张卡片，
.card: hover
{transform:rotateY(180deg);}

&、

过渡
transition: all 0.8s linear;

&、

透视(查看3D视图)
perspective: 800px;

&、

自定义字体(背面)

rotateY( )

2.翻转图片

步骤：

1、

构建一个无需列表，ul为整个图片框，ul内水平左浮动7个li，每个li显示图片的不同位置

2、

每个li内都构建俩div，一个为正面front，另个为反面back

正反两个div都隐藏背面，backface-visibility: hidden;

3、

li中(正反面父级)3D空间呈现，即：transform-style: preserve-3d;

4、

鼠标hover图片框ul 翻转li，ul: hover li{transform:rotateY(180deg);}

&、

翻转元素li的过渡 transition: all 0.8s linear;

&、

透视(查看3D视图)perspective: 600px;
