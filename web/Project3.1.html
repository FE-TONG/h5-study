CSS3~~animation


动画

animation: 动画名称 持续时间；

动画规则

@keyframes slide { /*动画规则 动画名称*/

from { 属性 : 值 ; } /* 从{ .... } */

to { 属性 : 值 ; } /* 到{ .... } */

}

播放次数

animation-iteration-count：n (次数) | infinite (无限循环) ;

暂停播放

animation-play-state：paused ;

合并语法

animation: name duration timing-function delay iteration-count direction ;


无缝滚动

步骤：

1、构建一个盒子box为图片框，宽度600px，overflow: hidden ;

2、在box中构建一个ul，宽度足够长，并定义动画属性，无限循环

3、ul中构建8个li水平左浮动，每4个为一组，两组图片一样，box可视li为3个

这样当第1个li往左边滚出，第5个li刚从右边滚入，这两个li放置的图片相同，实现无缝滚动

4、@keyframes slide

{

from{ transform : translateX( 0px ) ; }

to{ transform : translateX( -808px ) ; }

}

每个li 202px宽(包括border)，往左滚动4个li，所以translateX( -808px ) ;

5、鼠标移入暂停动画：.box:hover ul { animation-play-state : paused ; }

6、暂停后被选中的图片动画：ul li : hover{ animation : photo 1s ; }

@keyframes photo

{

0%, 20%, 50%, 80%, 100% { transform: translateY(0); }

40% { transform: translateY(-30px); }

60% { transform: translateY(-15px); }

}
