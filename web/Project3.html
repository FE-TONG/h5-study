CSS3~~animation


动画

animation: 动画名称 持续时间；

动画规则

@keyframes slide { /*动画规则 动画名称*/

from { 属性 : 值 ; } /* 从{ .... } */

to { 属性 : 值 ; } /* 到{ .... } */

}

播放次数

animation-iteration-count：n (次数) | infinite (无限循环) ;


无缝滚动

步骤：

1、构建一个盒子box为图片框，宽度660px，相对定位，overflow: hidden ;

2、在box中构建一个ul.list,宽度足够长，相对box绝对定位，并定义动画属性，无限循环

3、ul中构建10个li水平左浮动，每5个为一组，两组图片一样，box可视li为4个

这样当第1个li往左边滚出，第6个li刚从右边滚入，这两个li放置的图片相同，实现无缝滚动

4、@keyframes slide

{

from{ left: 0px ; }

to{ left: -800px ; }

}

每个li 160px宽(包括border)，往左滚动5个li，所以left: -800px ;

倒影，X轴旋转就可以了，transform: ratateX(-110deg); 再加透视效果
