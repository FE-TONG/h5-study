横竖屏响应式

代码优化：

1.响应式布局因为要准备几套css样式，因此在布局的时候会有大量代码冗余，如果不进行代码优化，网站到后期，代码越来越多，网站就很臃肿，影响浏览器在不同分辨率屏幕下对不同css样式的渲染速度，也就影响网站的打开速度。

2.在这几套css样式中查找相同的样式属性，然后提出@Media screen and ... { } 之外作为公共样式。

3.这些重复的样式属性比如：

模块background

main居中显示

header、footer的高度

header、container、footer的margin-top

left、center、right的高度
